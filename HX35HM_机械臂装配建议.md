## HX‑35HM 舵机机械臂装配建议

本说明针对你当前使用的 HX‑35HM 总线舵机 + STM32 控制板 + SO101 单臂（follower），目的是在**机械装配阶段一次性把“机械零位”和软件零位对齐**，减少后期标定和撞机风险。

---

### 1. 装配前的准备

#### 1.1 确认舵机与控制板工作正常

在装到机械臂上之前，完成以下检查：

1. 所有舵机单独接线、确认供电正常（电压 9.6–12.6 V，电源有足够电流余量）。
2. 使用控制板 + `continuous_sweep.py` 或上位机软件：
   - 验证每个舵机能正常转动、无卡滞、无异常噪声；
   - 确认方向一致（正方向定义清晰，例如：从电机线朝向你看，逆时针为正）。
3. 使用 `change_servo_id.py`，为每个舵机设置唯一 ID（例如 1~6），并记录好 **ID ↔ 关节位置** 的计划对应关系。

#### 1.2 规划关节与舵机 ID 映射

建议事先在纸上或文档里写明：

- ID1 → `shoulder_pan`（底座旋转）
- ID2 → `shoulder_lift`（大臂抬升）
- ID3 → `elbow_flex`（肘关节）
- ID4 → `wrist_flex`（腕俯仰）
- ID5 → `wrist_roll`（腕滚转）
- ID6 → `gripper`（夹爪）

并据此调整 `JOINT_ID_MAP`，与实际安装完全一致，避免后期逻辑混乱。

---

### 2. 舵机中位与“机械零位”定义

#### 2.1 理解 HX‑35HM 的中位

根据说明书（默认配置）：

- 位置范围（舵机模式）：
  - `pos = 0`   → `angle ≈ 0°`
  - `pos = 500` → `angle ≈ 120°`（中位）
  - `pos = 1000`→ `angle ≈ 240°`
- 控制角度范围可以扩展到 0–1500（0–360°），但建议在机械结构允许范围内留安全裕量。

在装配时，需要选定一个**机械基准姿态**，并约定它对应的“软件零位”或中位，比如：

- 肩关节（`shoulder_pan`）：
  - 机械臂从正前方看时，基座正对前方作为 0°。
- 肩抬升（`shoulder_lift`）：
  - 上臂自然下垂或水平伸出作为 0°。
- 肘关节（`elbow_flex`）：
  - 手臂伸直作为 0°，向上弯曲为正。
- 腕关节（`wrist_flex` / `wrist_roll`）：
  - 工具坐标系与手臂对齐的自然姿态作为 0°。
- 夹爪（`gripper`）：
  - 半开或默认张开姿态作为参考位置。

#### 2.2 装配前先把舵机调到基准位置

推荐做法：

1. 对每个舵机，使用控制板脚本或上位机软件，让舵机转到指定位置：
   - 若用中位作为机械零位，则先将舵机打到 `pos ≈ 500`；
   - 若用 0° 作为机械零位，则打到 `pos ≈ 0` 或厂商推荐的装配位置。
2. 在舵机保持通电、不强扭的情况下进行安装：
   - 此时舵盘的角度就是你定义的零位或中位；
   - 在这种姿态下装上连杆或关节支架。

这样可以保证：**软件认为的 0°/中位，与机械安装时的实际姿态一一对应**。

---

### 3. 每个关节的装配步骤示例

以下以常见 6 自由度机械臂为例（可按你的结构微调）：

#### 3.1 底座旋转（shoulder_pan，ID1）

1. 将 ID1 舵机打到中位（`pos=500` 或预设零位）。
2. 机械臂底座固定在平台上，舵机轴与底座同心。
3. 将连接上臂的旋转平台放置在“正前方”方向，对齐好中心线。
4. 在此位置锁紧舵盘螺丝。

> 结果：发送 `shoulder_pan = 0 rad` 时，机械臂基座正对前方。

#### 3.2 肩抬升（shoulder_lift，ID2）

1. 将 ID2 舵机打到中位（或期望零位）。
2. 固定舵机在底座/肩关节支架上，使舵轴垂直于旋转平面。
3. 将上臂段摆放到期望的零姿态：
   - 例如：水平伸出或自然下垂（根据你的 URDF 定义）。
4. 对齐上臂和舵盘之间的连接孔位，并在此姿态下锁紧。

> 结果：`shoulder_lift = 0 rad` 时，上臂处在你预期的参考姿态。

#### 3.3 肘关节（elbow_flex，ID3）

1. 将 ID3 舵机打到中位。
2. 将前臂与上臂串联，在“手臂伸直”的姿态下连接：
   - 确保前臂无扭曲，关节中心对齐。
3. 锁紧连接处的舵盘和轴。

> 结果：`elbow_flex = 0 rad` 表示手臂完全伸直，正向弯曲为正值。

#### 3.4 腕关节（wrist_flex / wrist_roll，ID4/5）

1. 将 ID4/ID5 舵机分别调到所需零位（通常是中位）。
2. 确保末端执行器（安装板 / 手爪座）的 Z 轴与前臂轴线方向匹配：
   - `wrist_flex = 0`：工具坐标系与前臂保持对齐；
   - `wrist_roll = 0`：夹爪“面朝下”或“面朝前”等一个统一姿态。
3. 根据 URDF 的坐标定义来选取上述基准姿态，以免后续 TF/MoveIt 表现反直觉。

#### 3.5 夹爪（gripper，ID6）

1. 将 ID6 舵机打到一个安全的中开位置：
   - 避免完全闭合或完全张开的位置作为零位；
   - 例如：pos=500 作为“半开”状态。
2. 在此位置装上夹爪连杆：
   - 确保夹爪在零位时两指对称；
   - 确保闭合时不会机械干涉或过度挤压。

### 4. 偏差与限位的精细调节

#### 4.1 舵机内偏差（offset）校正

装配完成后，如果某个关节在零位姿态下有少量偏差：

1. 使用上位机软件或控制板 SDK：
   - 读取当前偏差；
   - 在**中位时**微调 offset（例如 ±几度）；
   - 将 offset 写入舵机 EEPROM。
2. 避免在软件（ROS/MoveIt）中做大角度的偏差补偿，尽量在舵机层面解决。

#### 4.2 舵机角度限位

为防止机械撞击，建议在 HX‑35HM 中设置安全角度范围：

1. 根据机械结构，计算各关节允许的最小/最大角度；
2. 使用控制板 SDK / 上位机软件设置：

   - `bus_servo_set_angle_limit(servo_id, [min_pos, max_pos])`

3. 再在 URDF 中相应设置关节 `limit`，确保 MoveIt 规划不会超出这些范围。

---

### 5. 软件与机械对齐检查

装好整臂后，建议按以下步骤验证：

1. 启动单臂 HX‑MoveIt 栈：

   ```bash
   cd ~/ros2_ws
   source install/setup.bash
   ros2 launch so101_bringup follower_hx35hm_moveit.launch.py use_sim_time:=false
   ```

2. 在 RViz 的“Joint Space”界面中，将所有关节设为 `0 rad`（或你定义的零位角）；
3. 执行一次“Plan and Execute”，观察实际机械臂是否：
   - 基座朝向正确；
   - 各段姿态与 URDF 显示一致；
   - 无明显偏差。

如存在偏差，可优先在舵机 offset 中做小幅调整，再必要时更新 URDF 中关节零点和限位。

---

### 6. 安全建议

1. **初期测试时降低速度和行程**
   - 在 HX‑HM 中将最大速度、加速度参数调得保守一些；
   - MoveIt 中尽量不要一次规划到极限姿态。
2. **逐关节测试**
   - 每次只启用一个关节测试其方向、幅度与限位是否合理，再逐步联调多关节动作。
3. **始终保证供电质量**
   - 电压稳定、线径充足并使用合适的保险和滤波电容，防止电压跌落导致舵机复位或控制板异常。

通过以上装配和标定流程，可以最大限度地保证 HX‑35HM 舵机与 SO101 机械臂和 ROS2/MoveIt 软件栈在“零位、方向和行程”上的一致性，为后续的 teleop 和策略执行打下稳定基础。\
