# ros2_ws

基于 ROS2 的机械臂集成与开发工作空间，包含源码、控制脚本、标定数据和相关说明文档。

## 主要内容

- `src/`：ROS2 功能包源码
- `continuous_sweep.py`：扫描相关脚本
- `return_to_home.py`：机械臂回初始位脚本
- `aruco_handeye_*.json`：手眼标定数据与结果
- `*.md`：集成说明、装配建议和修改日志

## 环境

- Ubuntu 24.04
- ROS2

## 说明

本仓库仅跟踪源码、配置、说明文档和必要数据文件。  
`build/`、`install/`、`log/` 等编译与运行产物已忽略。
